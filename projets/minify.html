Minify

Un script pour minifier les fichiers

Voici mon module NPM pour minifier n'importe quel fichier.
Il est développé avec Node.js et est accessible via votre terminal.

...

À propos du module

Le module est très simple, il prend en compte les fichiers que vous lui donnez et les minifie. Il sait minifier les fichiers CSS, JavaScript, HTML, etc. Vous pouvez l'utiliser pour minifier vos fichiers avant de les déployer.

Vous pouvez l'utiliser en ligne de commande avec la commande `npm install @20syldev/minify.js` et en l'utilisant dans vos scripts. Il est très léger et rapide, il est donc parfait pour minifier vos fichiers rapidement lors de vos déploiements.
