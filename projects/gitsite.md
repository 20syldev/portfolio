## À propos {#about}

Ma plateforme d'hébergement web abordable, idéale pour les personnes ne maîtrisant pas l'hébergement de sites.
Avec des tarifs compétitifs, GitSite simplifie la mise en ligne de votre site grâce à un processus intuitif.

## Fonctionnalités {#features}

Vous pouvez héberger votre site sur un sous-domaine personnalisé comme `gitsite.dev/votre-nom`, offrant une adresse web professionnelle et mémorable.
Des tarifs adaptés à tous les budgets. Un site en ligne 24h/24 et 7j/7. Hébergez des blogs, portfolios ou autres !

## Processus d'hébergement {#process}

Le processus d'hébergement se fait en **4 étapes** simples via GitHub :

1. **Fork** : Clonez le dépôt GitSite sur votre compte GitHub
2. **Modification** : Ajoutez ou uploadez vos fichiers de site
3. **Soumission** : Validez vos modifications et créez une pull request
4. **Validation** : L'équipe examine et accepte votre demande

## Derrière le projet {#stack}

GitSite est un des premiers sites de vente que j'ai créé. Le système de contact est clair, et il est facile de comprendre comment fonctionne la procédure d'hébergement.
