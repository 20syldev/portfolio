## Fonctionnalités {#features}

Mon API est hébergée sur [api.sylvain.sh](https://api.sylvain.sh) et elle est disponible 24/7.
Développée avec Node.js, elle propose de nombreux modules spécialisés couvrant une grande variété de besoins, comme la génération de QR codes, de couleurs ou de tokens sécurisés, mais aussi un système de chat temporaire et un jeu de morpion jouable via l'API.

Vous pouvez accéder à chaque service facilement en effectuant des requêtes HTTP avec des paramètres ajustables.
Elle est conçue pour être simple à utiliser et facile à intégrer dans vos projets. L'API repose sur une architecture REST, vous permettant d'interagir avec les services via des requêtes HTTP standards.

La liste complète des modules et leurs paramètres est disponible sur la [documentation](https://docs.sylvain.sh).
L'API est aussi disponible en tant que [module NPM](https://npmjs.com/package/@20syldev/api) pour une intégration directe dans vos projets Node.js.

## Qu'est-ce qu'une API REST ? {#rest}

Une API REST permet à deux applications de communiquer entre elles en exposant des ressources via des URL fixes et des méthodes HTTP (GET, POST, PUT, DELETE). Ce style d'architecture rend l'intégration facile et accessible, sans nécessiter une complexité inutile.

## Pourquoi utiliser cette API ? {#why}

Cette API est idéale pour répondre rapidement à vos besoins de développement. Que vous ayez besoin de générer un QR code, de créer un token ou d'utiliser des algorithmes, elle est là pour vous offrir une solution simple et efficace à intégrer dans vos projets.

## Commencez dès maintenant {#start}

Vous êtes prêt à intégrer l'API dans votre projet ? La documentation complète est disponible sur [docs.sylvain.sh](https://docs.sylvain.sh), où vous trouverez des descriptions détaillées de chaque endpoint, ainsi que des exemples de requêtes et de réponses.

## Limites d'utilisation {#limits}

L'API est gratuite et met à votre disposition un quota de 2000 requêtes par heure !
Vous pouvez augmenter la limite de requêtes en vous rendant sur la page [pricing](https://docs.sylvain.sh/v3/fr/pricing) de la documentation, et en achetant un jeton d'API.
