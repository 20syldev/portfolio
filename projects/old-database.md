## À propos {#about}

Ce projet était une interface de base de données serverless que j'avais développée.
Il permettait de stocker et gérer des données directement depuis le navigateur, sans nécessiter de serveur backend dédié.
Le site a été archivé et n'est plus disponible en ligne.

## Fonctionnalités originales {#features}

L'interface proposait une gestion simplifiée des données avec une approche sans serveur.
L'objectif était d'offrir une solution légère pour stocker et consulter des données sans avoir besoin de configurer un serveur ou une base de données traditionnelle.
Le projet exploitait les technologies web standards (HTML, CSS, JavaScript) pour offrir une expérience utilisateur fluide et accessible.

## Code source {#code}

Le code source reste accessible sur [GitHub](https://github.com/20syldev/old-database) pour consultation et apprentissage.
Vous pouvez copier le code et essayer de le comprendre pour créer votre propre base de données sans serveur.
