## À propos {#about}

WrkIT est un bot pour améliorer l'expérience sur le serveur Discord du BTS SIO.
Il offre quelques fonctionnalités utiles permettant d'automatiser certaines tâches et simplifier la gestion.

## À propos du bot {#features}

WrkIT est un bot Discord simple qui permet de gérer quelques fonctionnalités utiles pour le serveur du BTS SIO.
Il offre des commandes pour afficher l'emploi du temps, ou gérer les événements grâce à des commandes.
Il est disponible en permanence, offrant une expérience constante et sans interruption.

## Création {#creation}

WrkIT a été créé en quelques heures pour automatiser certaines tâches sur le serveur Discord du BTS SIO.
Il a été conçu pour être modulable, permettant ainsi d'ajouter de nouvelles fonctionnalités facilement.

Le bot est développé en JavaScript et utilise la bibliothèque Discord.js pour communiquer avec l'API de Discord.
Il est hébergé sur un serveur dédié pour garantir une disponibilité constante.
Il utilise aussi mon API personnelle pour récupérer les données de l'emploi du temps.
