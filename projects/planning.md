## À propos {#about}

Cette interface web est un planning pour les SLAM et SISR de Ensitech.
Elle est disponible sur [planning.sylvain.pro](https://planning.sylvain.pro).

## À propos du planning {#features}

J'ai conçu cette interface dans le but de simplifier la consultation des emplois du temps pour les étudiants.
Le planning récupère les données directement depuis HyperPlanning et les affiche de manière claire et organisée.
L'interface se veut moderne et intuitive, permettant de naviguer facilement entre les jours et les semaines.

## Une navigation fluide {#navigation}

L'interface propose une navigation fluide entre les jours via le swipe sur mobile ou le scroll et les flèches sur PC.
Vous pouvez également afficher une vue complète des cours groupés par semaine.
Un indicateur en temps réel met en évidence le cours actuel pour savoir rapidement où vous en êtes.

## Des fonctionnalités pratiques {#features-2}

L'application détecte automatiquement la pause déjeuner et affiche les détails des cours (professeur, salle, classes, notes) en mode déroulant.
Vos préférences de spécialité et de vue sont sauvegardées localement, vous n'avez donc pas besoin de les reconfigurer à chaque visite.

Le planning se rafraîchit automatiquement toutes les 5 minutes pour vous garantir des informations à jour.
L'interface est optimisée aussi bien pour mobile que pour PC, afin que vous puissiez consulter votre emploi du temps où que vous soyez.
