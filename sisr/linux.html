Linux Mint - Installation de la distribution

Guide complet pour l'installation de Linux Mint, version Cinnamon.

Installation de Linux Mint (Cinnamon Edition)

Suivez les étapes ci-dessous pour installer Linux Mint avec l'environnement de bureau Cinnamon sur votre PC :

  • Téléchargez l'image ISO de Linux Mint Cinnamon Edition depuis le site officiel de Linux Mint.
  • Gravez l'image ISO sur une clé USB ou un DVD à l'aide de Rufus (pour Windows) ou dd (pour Linux).

Une fois la clé USB prête, redémarrez votre PC et entrez dans le BIOS/UEFI pour démarrer à partir de la clé USB ou du DVD.

Sélectionnez "Démarrer Linux Mint" pour lancer l'installateur.

Configurer le système

Une fois que le système est démarré, vous serez accueilli par l'écran d'installation. Voici les étapes suivantes :

  • Sélectionnez votre langue (français, anglais, etc.).
  • Cliquez sur "Installer Linux Mint".

Choisissez votre disposition de clavier. Vous pouvez également tester le clavier avant de continuer.

Choisir une disposition : Français (variante)

Choisissez ensuite votre type d'installation. Si vous avez une partition vierge, sélectionnez "Effacer le disque et installer Linux Mint". Si vous souhaitez configurer les partitions manuellement, sélectionnez "Autre chose".
Si vous optez pour l'installation automatique, le programme partitionnera le disque en fonction de l'option choisie (type de partitionnement recommandé).

Exemple de partitionnement pour une installation automatique :

/dev/sda1 - Partition principale (ext4)
/dev/sda2 - Partition de swap

Si vous souhaitez personnaliser les partitions, voici un exemple de configuration :

/dev/sda1 - Partition principale, formatée en ext4
/dev/sda2 - Partition swap

Installation du Système

Une fois le partitionnement configuré, cliquez sur "Installer maintenant". Le programme d'installation vous demandera de confirmer les modifications apportées aux partitions.

Installation du système en cours...

Le processus d'installation va se poursuivre, et vous devrez répondre à quelques autres questions :

  • Choisissez votre fuseau horaire.
    • Choisir la localisation : Europe/Paris
  • Créez un utilisateur avec un nom et un mot de passe.
    • Utilisateur : Sylvain
    Mot de passe : ********

    Si vous voulez activer l'authentification automatique (sans avoir à entrer le mot de passe à chaque démarrage), vous pouvez cocher l'option "Se connecter automatiquement".

Une fois ces étapes terminées, l'installation va se poursuivre et finaliser l'installation des fichiers.

Installation des fichiers du système...

À la fin du processus, vous serez invité à redémarrer le PC.

Redémarrage... Veuillez retirer la clé USB/DVD et appuyer sur "Entrée".

Post-Installation et Configuration Initiale

Une fois que Linux Mint a démarré, voici les étapes pour finaliser l'installation et la configuration :

  • Mettez à jour le système pour vous assurer que tout est à jour :
    • $ sudo apt update
$ sudo apt upgrade
    Reading package lists... Done
Building dependency tree
Reading state information... Done
The following packages will be upgraded:
  mint-meta-cinnamon ...
xx packages upgraded, xx newly installed, xx to remove and xx not upgraded.
  • Installez les pilotes recommandés pour votre matériel :
    • $ sudo apt install mintdrivers
    Reading package lists... Done
Building dependency tree
Reading state information... Done
The following additional packages will be installed:
  linux-headers-5.x.x-xx linux-image-5.x.x-xx
  • Redémarrez le PC pour appliquer tous les changements :
    • $ sudo reboot
    [sudo] password for sylvain: ***************

Après le redémarrage, Linux Mint avec Cinnamon sera complètement installé et configuré.
